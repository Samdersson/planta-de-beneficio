este sisema tiene como funcionalidad gestionar y llevar contro de los procesos realizados en un aplanata de beneficio animal para bovinos y porcinos
de manera que se encarga de gestionar desde el ingreso de las guias de movilizacion, crear ususarios conn sus respectivos roles, crear clientes, crear productores.
permite ver los informes por rangos de fechas para obtener un control mas detallado del sacrificio de animales.

el sistema esta desarrollado bajo el leguaje PHP como back, html y css para el front y para realizar validaciones por medio de JavaScript al ser desarrollado sin frameword
se pude instalar en equipos de computo con pocos recursos y sin complicaciones.

pasos para la instlacion del sistema.
1 tener instalado xampp version 8.2 o superior
2 crear un nuevo esquema de base de datos con el nombre prueba_software luego importa el archivo SQL 
3 descargar la copia del repositorio en la carpeta htdocs

para iniciar el el uso del sistema ve a la ruta http://localhost/planta_de_beneficio/front/inicio.html
